```
====================================================================
#                                                                  =
#                           *******                                =
#                              **                                  =
#                         ****  **                                 =
#                       ******   **                                =
#                     ******      ***                              =
#                     ****      **  **                             =

#                                                                  =
#    ########   ##    ##  ########  ########  ########  ##    ##   =

#    ##         ########  ########  ########  ########  ########   =
#    ##         ##    ##  ##        ##   ##   ##   ##      ##      =
#    ########   ##    ##  ########  ##    ##  ##    ##     ##      =
#                                                                  =
#                                                                  =
#                                                                  =
====================================================================
```
